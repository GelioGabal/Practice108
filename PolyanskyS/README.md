 Отчет по работе 108. Secure CI

## Описание работы
Настроен CI-процесс с проверками безопасности для приложения генератора QR-кодов.

## Инструменты безопасности

| Инструмент | Назначение |
|------------|------------|
| Bandit | Статический анализ Python кода |
| Gitleaks | Поиск секретов в репозитории |
| Trivy | Сканирование Docker образов |
| CodeQL | Продвинутый статический анализ |
